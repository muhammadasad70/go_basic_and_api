# Go REST API

A backend API built using Golang and Gin Framework.

## Features

- User Registration
- Login Authentication
- JWT Authorization
- CRUD Operations
- PostgreSQL Integration

## Tech Stack

- Golang
- Gin
- PostgreSQL
- JWT
- Docker

## API Endpoints

### Authentication

POST /signup

POST /login

### Users

GET /users

PUT /users/:id

DELETE /users/:id
